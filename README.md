# Sales Analysis Dashboard - Power BI

## 📊 Project Overview
This dashboard analyzes sales performance across products, territories, and order status to provide clear business insights.

## 🛠 Tools Used
- Power BI
- DAX
- Excel Dataset

## 📈 Key KPIs
- Total Sales: 60M+
- Total Orders: 1.47K
- Total Quantity: 171K+
- Total Tax: 2.93M
- Average Freight: 38.81

## 📊 Dashboard Features
- Sales analysis by Product Category
- Orders distribution by Status
- Sales trend by Order Date
- Territory performance comparison
- Interactive slicers and filters

## 🔎 Insights
- Bikes category generates the highest sales.
- Canada leads in total sales by territory.
- Most orders are in Approved status.
- Sales trend shows growth over time.
